# Chapitre 7 : Web Analytics & Data

Si vous ne le mesurez pas, vous ne pouvez pas l'améliorer.

## Google Analytics 4 (GA4)
L'outil standard pour comprendre le comportement des utilisateurs.
- **Acquisition** : D'où viennent les visiteurs ?
- **Engagement** : Que font-ils sur le site ?
- **Rétention** : Reviennent-ils ?

## Les indicateurs clés (KPIs)
- **Taux de conversion** : % de visiteurs qui achètent.
- **Taux de rebond** : % de visiteurs qui partent immédiatement.
- **Valeur vie client (LTV)** : Revenu généré par un client sur le long terme.

## Tests A/B
Comparer deux versions d'une page pour voir laquelle performe la mieux.
