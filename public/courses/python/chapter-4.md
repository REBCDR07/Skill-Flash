# Fonctions et Modules

## Fonctions

Une fonction est un bloc de code réutilisable.

```python
def ma_fonction(nom):
  print("Bonjour " + nom)

ma_fonction("Alice")
```

## Modules

Vous pouvez importer des fonctions depuis d'autres fichiers ou bibliothèques.

```python
import math
print(math.pi)
```
