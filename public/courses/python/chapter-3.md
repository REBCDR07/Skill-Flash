# Les Boucles et Conditions

## Conditions (If/Else)

```python
age = 18
if age >= 18:
    print("Majeur")
else:
    print("Mineur")
```

## Boucles (Loops)

### While

```python
i = 1
while i < 6:
  print(i)
  i += 1
```

### For

```python
fruits = ["pomme", "banane"]
for x in fruits:
  print(x)
```
