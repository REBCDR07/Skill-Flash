# Chapitre 10 : Bonnes Pratiques & Performance

Écrire du code qui fonctionne est bien, écrire du code performant et maintenable est mieux.

## Clean Code
- Noms de variables explicites.
- Fonctions courtes avec une seule responsabilité.
- Évitez les "magic numbers".

## Optimisation de la Performance
- **Debouncing & Throttling** : Limiter la fréquence d'appel des fonctions (ex: scroll).
- **Lazy Loading** : Charger les données uniquement quand nécessaire.
- **Memory Leaks** : Nettoyez les `addEventListener` ou `setInterval` non utilisés.

## Tests Unitaires
Indispensables pour garantir que votre code ne se casse pas lors de modifications.
- Jest
- Vitest

## Conclusion du Cours
Vous maîtrisez maintenant les bases solides du JavaScript moderne. La prochaine étape est de pratiquer sur des projets réels ou d'explorer des frameworks comme React ou Vue.js.
