# Chapitre 9 : SEO et Meta Tags

Le SEO (Search Engine Optimization) commence dans la balise `<head>` de votre document HTML.

## 1. La balise Title

C'est l'élément le plus important pour les moteurs de recherche. Il apparaît dans l'onglet du navigateur et les résultats Google.

```html
<title>Cours HTML Complet - SkillFlash Academy</title>
```

## 2. Les Meta Descriptions

Elles fournissent un résumé court de votre page.

```html
<meta name="description" content="Apprenez les bases et les techniques avancées du HTML5 en 10 chapitres interactifs.">
```

## 3. Open Graph (Partage Social)

Pour contrôler comment votre site s'affiche sur Facebook, LinkedIn ou Twitter/X.

```html
<meta property="og:title" content="Devenez expert HTML">
<meta property="og:image" content="https://site.com/preview.jpg">
```

## 4. La balise Cannonical

Évite le contenu dupliqué.

```html
<link rel="canonical" href="https://site.com/cette-page">
```
