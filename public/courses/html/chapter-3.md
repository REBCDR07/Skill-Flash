# Chapitre 3 : Texte et Typographie

Le texte est le cœur de toute page web. HTML fournit une variété de balises pour structurer et donner du sens à votre contenu textuel.

## 1. Titres (h1 à h6)

Les titres sont essentiels pour la hiérarchie de votre page. Ils vont de `<h1>` (le plus important) à `<h6>` (le moins important).

```html
<h1>Titre de niveau 1</h1>
<h2>Titre de niveau 2</h2>
<h3>Titre de niveau 3</h3>
```

## 2. Paragraphes et Blocs

La balise `<p>` définit un paragraphe. Pour les citations longues, utilisez `<blockquote>`.

```html
<p>Ceci est un paragraphe standard qui contient du texte.</p>

<blockquote>
  "Le web est pour tout le monde." - Tim Berners-Lee
</blockquote>
```

## 3. Mise en forme de base

- `<strong>` : Pour le texte important (affiche en gras).
- `<em>` : Pour mettre l'accent (affiche en italique).
- `<br>` : Pour un retour à la ligne forcé.
- `<hr>` : Pour une ligne de séparation thématique.

## Exemple complet

```html
<h1>Bienvenue sur SkillFlash</h1>
<p>Apprenez le <strong>HTML</strong> rapidement et <em>efficacement</em>.</p>
<hr>
<h2>Pourquoi apprendre le HTML ?</h2>
<p>C'est la base de tout ce que vous voyez sur le web aujourd'hui.</p>
```
